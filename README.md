##easySlide

###一、What is easySlide?

easySlide is one jquery plugin for slider. it is easyer to use and extend.
###二、How to use
$("#easySlide").easySlide();
####or 
$("#easySlide").easySlide({
  slideTime:700,	//动画滑行速度，越大越慢
  autoPlay:true,	//true为自动播放，
  pauseTime:3000,	//动画暂停时间
  hoverPause:false, //是否鼠标悬停,默认为false
  index:0 //展示项目的索引
});
